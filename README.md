allocine
============

A module to use Allocine API V3 in Python.

More details on the API usage at http://wiki.gromez.fr/dev/api/allocine_v3

A PHP version is available at https://github.com/gromez/allocine-api


Usage
-------

    $ python test_api.py

Documentation
-------

    $ pydoc allocine
